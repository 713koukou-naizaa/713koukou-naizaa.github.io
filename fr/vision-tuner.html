Vision Tuner

Cette application Python a été faite pour en apprendre plus sur le traitement d'images et divers algorithmes comme nearest neighbor, interpolation bilinéaire, le ré-échantillonnage de Lanczos. C'est pourquoi les algorithmes utilisés dans le programme sont non les algorithmes offerts par les bibliothèques Python comme cv2 mais des implémentations propres que j'ai faites après les avoir étudiés.

Nearest Neighbor

Nearest Neighbor est le premier algorithme que j'aie implémenté. J'ai commencé avec une version qui me paraissait lente, mais une fois que j'ai essayé de l'optimiser, j'ai réalisé que je pouvais éviter certains calculs dans des boucles imbriquées. La première capture d'écran est la version avant optimisation, la deuxième est la version post-optimisation.

Symétrie

La symétrie d'une image est la seconde fonction implémentée, cette fois-ci j'ai pu éviter des calculs dès le début grâce à mon expérience avec l'algorithme Nearest Neighbor. J'ai implémenté la symétrie verticale, horizontale et verticale + horizontale. Les algorithmes sont essentiellement les mêmes pour toutes les symétries. Voici la symétrie verticale.
