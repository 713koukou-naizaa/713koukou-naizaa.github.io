Memoris: Recollection

Memoris: Recollection est un jeu fantaisique RPG sur lequel je travaille avec un ami depuis Mai 2024. C'était au moment de sa conception notre projet le plus ambitieux, cependant cela m'a permis d'acquérir des compétences dans de nombreux domaines:

  • C# avec Unity
  • Production de documents officiels
  • LaTeX
  • Git et GitHub

C# avec Unity

Ce projet m'a aidé à acquérir de l'expérience en C# : j'ai pu utiliser des interfaces, de l'heritage et le polymorphisme. J'ai également appris comment utiliser l'éditeur Unity : j'ai travaillé sur des scènes, des composants, j'ai créé des scripts utilisant UnityEngine et j'ai fait des animations.

Production de documents officiels

De part la grande quantité d'informations, nous avions besoin de documents organisés afin de suivre l'historique, le lore, les informations sur les personnages... Nous avons donc fait une recherche sur comment organiser les documents d'une jeu et quel serait un document de conception de jeu.

LaTeX

Quitte à rédiger de nombreux docuemnts, nous avons décide d'utiliser LaTex pour faire tous les documents de notre projet. Nous avons tout d'abord utilisé Overleaf en ligne mais plus tard nous avons changé pour Visual Studio Code quand nous avons décidé de versionner les documents sur GitHub.

Git et GitHub

Afin de partager nos projets, suivre les changements que nous faisions et de sauvegarder les versions de nos travaux, nous avons utilisé Git et GitHub. Au début, nous avons créé une branche conception dans laquelle nous avions mis tous les documents.
