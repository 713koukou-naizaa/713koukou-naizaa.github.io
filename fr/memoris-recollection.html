Memoris: Recollection

Memoris: Recollection est un jeu fantastique RPG sur lequel je travaille avec un ami depuis mai 2024. C'était au moment de sa conception notre projet le plus ambitieux et il m'offre des possibilités d'apprendre dans de nombreux domaines, tout en me faisant découvrir de nouvelles compétences au fur et à mesure de son avancement et des nouvelles tâches que nous devons effectuer.

Git and GitHub

Afin de partager nos projets, de suivre les changements que nous faisions et de sauvegarder les versions de nos travaux, nous avons utilisé Git et GitHub. Au début, nous avons créé une branche conception dans laquelle nous avions mis tous les documents, et nous avons créé un projet GitHub dans le but d'utiliser le backlog intégré.

Unity

Nous prévoyons d'utiliser Unity pour créer notre jeu. Bien que je possède déjà quelques bases, cela me permettra de les renforcer : je travaillerai sur des scènes, des composants, j'écrirai des scripts avec UnityEngine et je ferai des animations.

C#

Puisque nous utiliserons Unity, je gagnerai plus d'expérience avec C# : nous avons déjà pensé à comment implémenter des classes et nous avons prévu d'utiliser des interfaces, l'héritage et du polymorphisme (principalement pour les personnages, les joueurs, les monstres…).

LaTeX

Quitte à rédiger de nombreux documents, nous avons décidé d'utiliser LaTex pour rédiger tous les documents de notre projet. Nous avons tout d'abord utilisé Overleaf en ligne, mais plus tard, nous avons changé pour Visual Studio Code quand nous avons décidé de versionner les documents sur GitHub.
