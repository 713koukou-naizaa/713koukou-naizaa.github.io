Reimu

Reimu est un bot discord en Python qui récupère des données d'un fichier ics et offre diverses commandes discord pour les visualiser. Créer ce bot m'a permis d'en apprendre plus sur les fichiers ics et leur structure pour traiter leurs données, de pratiquer le versionning de code avec une branche par version ainsi que comment adapter un programme Python en module comme on le ferait pour créer une API.

Git and GitHub

J'ai utilisé git pour versionner les différentes versions du bot en les séparant dans différentes branches. GitHub m'a permis de travailler sur le projet quand je voulais sur n'importe quelle machine.

Python

J'ai utilisé Python pour créer le bot, j'ai utilisé la bibliothèque requests de Python pour obtenir le fichier ics et l'ai transformé en un objet Calendar pour pouvoir le traiter.

Discord.py

Discord.py est la bibliothèque offerte par Discord pour programmer un bot en Python. Bien que la syntaxe soit spécifique à cette bibliothèque, elle demande encore de comprendre les principes de programmation asynchrone, de programmation basée sur les événements et la POO.
