Book'n'Go

Book'n'Go est une application web designée depuis 0 pour un projet de conception et de gestion de projet. Nous deviens tout d'abord penser au but de l'application, définir les principales fonctionnalités, penser aux futures principaux utilisateurs de l'application, créer des diagrammes de cas d'utilisation et scénarios ainsi que des maquettes. Nous avons aussi organisé plusieurs réunions de groupe ainsi qu'avec notre tuteur et nous avons écrit plusieurs rapports.

Figma

Nous avons créé une équipe sur Figma afin de travailler ensemble sur les dizaines de mock-ups wireframe pour les différents scénarios que nosu avions imaginés.

GanttProject

Le groupe a utilisé GanttProject pour planifier le projet.

UML and Modelio

UML a été utilisé avec Modelio pour créer les différents diagrammes de cas d'utilisation et ainsi planifier les principales fonctionnalités.
