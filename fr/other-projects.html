Reimu
Description

Reimu est un bot discord développé avec Python et la bibliothèque discord-py et mis à disposition des étudiants de mon université. Il utilise une version modifiée du programme timetable que j'ai développé pour récupérer des données d'un fichier .ics sur le web et il envoie un message avec des informations sur l'emploi du temps de l'utilisateur discord. J'ai créé ce bot, car le front-end du site web de mon université affichant l'emploi du temps ne s'actualisait pas de manière consistante.
Repo GitHub privé, veuillez me contacter pour plus d'informations.
