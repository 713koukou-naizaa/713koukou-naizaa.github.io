Scripts
auto-recorder.sh
Description

Auto-recorder est un script bash qui capture l'écran de l'hôte avec flameshot, prend une photo avec ffmpeg toutes les 15 secondes et les envoie par e-mail. C'est une mesure de protection en cas de vol de l'ordinateur hôte. De plus, il envoie l'adresse IP, la localisation et le SSID du Wi-Fi.

db-backup.sh & db-restore.sh
Description

db-backup.sh a été fait pour être exécuté avec un cron job pour faire une sauvegarde d'une base de données mariadb avec mariadb-dump, tout en zippant la sauvegarde. db-restore.sh a été créé pour restaurer une sauvegarde de mariadb. Les deux processus sont loggués.

dl-yt-playlist.py
Description

dl-yt-playlist.py est un script Python qui requête l'URL d'une playlist YouTube et télécharge chaque vidéo de la playlist donnée en paramètre. Il utilise l'API YouTube Data API v3.

tp.sh
Description

J'aime garder mes fichiers organisés, mais je finis souvent avec beaucoup de dossiers, c'est pourquoi j'ai décidé de créer ce script. tp.sh est un script bash qui ouvre l'explorateur après avoir appelé le script avec des paramètres prédéfinis dans le terminal.

cleanDownloads.py
Description

clean-Downloads.py est un script Python que j'ai fait pour déplacer des fichiers de mon dossier Téléchargements vers un autre emplacement. Je préfère garder certains types de fichiers seulement dans mon dossier Téléchargements pour archiver ce que j'ai téléchargé, alors que j'ai tendance à supprimer le reste, c'est pourquoi j'ai choisi de faire ce script.

getAllFilesStats.ps1
Description

getAllFilesStats.ps1 est un script powershell que j'ai fait pour compléter cleanDownloads.py. Puisque je garde principalement les installeurs dans mon fichier Téléchargements, ce dernier a tendance à devenir très large, donc pour éviter de gaspiller de l'espace de stockage, mais continuer de les archiver, j'ai créé ce script qui récupère toutes les métadonnées des fichiers dans mon dossier Téléchargements et les écrit dans un fichier texte.
