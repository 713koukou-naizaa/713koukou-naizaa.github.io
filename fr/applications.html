Quest Log
Android Kotlin Jetpack Compose
Description

Quest Log est une application de productivité Android qui gamifie le suivi de tâches journalières à travers des méchaniques de RPG. Les utilisateurs complètent des quêtes quotidiennes et hebdomadaires pour gagner de l'expérience, monter de niveaux et éviter les pénalités, ce qui encourage les habitudes consistentes par une progression inspirée des jeux.

Reimu
Python Discord Bot Discord.py
Description

Reimu est un bot Discord développé avec Python et la bibliothèque discord-py, et mis à disposition des étudiants de mon université. Il utilise une version modifiée d'un programme d'emploi du temps en python que j'ai développé pour récupérer des données d'un fichier .ics depuis le web, et envoie des messages avec des informations sur l'emploi du temps en fonction du rôle de l'utilisateur Discord surun serveur Discord. J'ai créé ce bot car le frontend du site de mon université affichant l'emploi du temps n'était pas tout le temps mis à jour de manière consistante.
Dépôt Github privé, veuillez me contacter pour plus d'informations.
