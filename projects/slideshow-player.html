Slideshow player

This slideshow player was the project of one of the 6 skills I studied in my first year of Computer Science. We were given specifications about the application and we had to implement each of them in our application. Here is the link to the GitHub repository

Qt

We used Qt Creator as an IDE which got us to discover how projects were made in Qt. We also had the opportunity to work with Qt Designer which really simplified the process of creating the GUI. We used tools like Qt's widgets, layouts, buttons and text input fields link to functions with the help of signals and slots.

SQL

The pictures and their informations to display in the slideshow were stored in a database as a path. Therefore we used SQL libraries given by Qt to send requests to the database and retrieve or alter the targeted data.
