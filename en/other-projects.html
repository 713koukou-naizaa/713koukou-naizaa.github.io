Reimu
Description

Reimu is a discord bot developped with python and the discord-py library and put at the university students' disposal. It uses a modified version of the timetable python program I developed to fetch data from a .ics file on the web, and it sends messages with information on the timetable of a discord user. I created this bot because the front-end of the website of my university displaying the timetable didn't update consistently.
Github repository private, please contact me for more details.
