Quest Log

Quest Log is an Android productivity app that gamifies daily task tracking through RPG mechanics. Users complete daily and weekly quests to gain experience, level up, and avoid penalties, encouraging consistent habits through game-inspired progression.

Architecture

The project follows a MVVM architecture combined with a repository layer to separate the UI, the state, and the data management :

  • ViewModel manages the UI state and business logic
  • Repository layer abstracts data access
  • Jetpack Compose UI reacts to state changes
State updates are handled using Kotlin Coroutines and Flow, enabling reactive UI updates.

Data Management

User data (profile, progression, quest state) is persisted locally using DataStore. This system ensures :

  • Persistence across sessions
  • Consistent state updates
  • Lightweight local storage without a full database
