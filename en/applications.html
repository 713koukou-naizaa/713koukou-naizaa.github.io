Reimu
Python Discord Bot Discord.py
Description

Reimu is a Discord bot developed with Python and the discord-py library and put at the university students' disposal. It uses a modified version of a timetable Python program I developed to fetch data from a .ics file from the web, and it sends messages with information on the timetable based on the Discord user's roles in a Discord server. I created this bot because the front-end of the website of my university displaying the timetable didn't update consistently.
Github repository private, please contact me for more details.
