Presentations Scheduler
C++ Algorithm
Description

Presentations Scheduler is a C++ scheduling component implementing a greedy algorithm to assign student internship presentations to time slots. Designed for integration with a PHP web application, it prioritizes performance and efficiency through compiled execution.

Vision Tuner
Python Image Processing Algorithm
Description

Vision Tuner is a Python command-line tool I'm developing to learn image processing algorithms from scratch. The goal of the program is to allow its users to edit pictures via hand-made algorithms (nearest neighbor/bilinear interpolation, etc.).

Owl Eye
Python Unix Security Automation
Description

OwlEye is a monitoring tool that automatically takes webcam photos and screenshots at regular intervals as well as collecting system information such as the IP address, the MAC address, the WiFi SSID, and geolocation, and sends this data via email. Designed to run in the background, it can help secure your device or log activity for legitimate purposes.
