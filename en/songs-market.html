Songs market

Songs market is a web application selling songs online; it doesn't aim to be deployed and fully functional. Songs can be added to a cart, which can be viewed. It also contains a back-office page to add, remove, and modify songs.

HTML

HTML was used to create the front-end of the application.

JS

JS allowed me to create a modal to add a song to the cart.

PHP

I used PHP to manage the interactions with the database (back office), and also to manage the cart in session and the fake payment process.

SQL

SQL was used to interact with the database (back office and retrieve songs).

Git and GitHub

I used Git and GitHub to manage the project, apply version control, and share it with my teacher.

Composer

Composer allowed me to use the getid3 dependency to extract metadata from audio files (their duration).
