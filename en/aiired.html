Aiired

Aiired is a project aiming to emulate a cognitive system powered by locally hosted LLMs. It combines voice input/output, vector memory, and specialized agents to perform reasoning, memory retrieval, and task execution entirely on a local machine.

Current state

Currently, Aiired already handles voice input, transcription, response generation via a locally hosted LLM, and speech output. The system can retrieve and store timestamped memories across multiple layers, allowing it to maintain contextual and long-term conversations.

Planned improvements

  • Integrate emotion detection in speech input
  • Implement background reasoning (“internal thoughts”) for richer responses
  • Improve the memory extraction
  • Introduce dynamic system prompts to allow the AI to adjust its personality itself
