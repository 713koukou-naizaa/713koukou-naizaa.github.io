Slideshow player
Used tools
  • Qt was used as the IDE and GUI creator
  • SQL was used in this project to send requests to a database
  • C++ was the main language
  • Git/GitHub as the version control tool
Description

This slideshow player was made to practice C++ coupled with Qt, Git/GitHub, SQL requests in C++ and HMI design. For this project we used QtCreator and QtDesigner, retrieved and altered data from a database using SQL requests with Qt's database libraries, and created a graphical user interface using Qt's widgets and layouts. Finally, Git/GitHub helped to version the project and share it with our coworker and teacher
