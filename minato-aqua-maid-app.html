Minato Aqua Maid App

Minato Aqua Maid App is a python app I made to discover TKinter and GUI programming in general. It consists in a simple to-do list saved locally in a txt file, and a random sound player.

Python

I used python and diverse libraries: os for the to-do list file, threading in order to play the random sounds in background and pygame to play the sounds. This project was also an occasion for me to learn about the general structure of python files using a modules folder, a core.py file, the __init__.py file and I created a small utility.py file in order to define specific methods I couldn't find online.
